Projects

Some of the things I've dabbled on.

Pall - Color Picker

Description

Pick a color. Anywhere. Open-source and cross-platform. Copies selected color to clipboard and saves it. Published to the Snap Store for Linux.

Caltax - Tax Calculator

Description

A simple web application that lets PH users calculate their taxes, social security, hdmf and philhealth. This also lets users peek at their take-home pay.

GimmeGifbar

Description

An open source, cross-platform desktop application that lets you search for a gif, copy its link or simply share it, right from the menubar. Using Electron framework, written in Javascript with Giphy-Api and Menubar modules.

Emgu CV Face Detection App for WPF

Description

A simple Face Detection app for the WPF platform that aims to be a "base" app for any face detection app you can think of. The app is written with MVVM pattern for scalability and maintainability, if ever you want to scale it up.
